## Setup

1. `git clone https://github.com/ElixirTechCommunity/Nexus_Website.git`
2. `open project in vscode.`
3. `create your personal branch.` 

### For development

1. `open dev.html.`
2. `start live server.`

### For Production

1. `open terminal and cd to project dir/`
2. run 
    - `$ npm run build`
3. `open index.html in vsCode.`
4. `start live server.`
